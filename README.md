# 🛡️ SYP Shield AI

Anti-Counterfeit Banknote Detection System

## Tech Stack
- Django 6.0
- Django REST Framework
- PyTorch (DeepCAE)
- JWT Authentication
- SQLite (Development)

## Setup
```bash
git clone https://github.com/username/syp_shield.git
cd syp_shield
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
